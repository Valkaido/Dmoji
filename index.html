Comment utiliser le bot

Comment écrire une émote

Tu peux utiliser l'un de ces formats :

  • /KEKW
  • /KEKW:1x
  • /KEKW:2x
  • /KEKW:3x
  • /KEKW:4x

Tu peux aussi écrire du texte normal dans le même message, par exemple Salut /KEKW.

Ce qui fonctionne

  • tu envoies une émote seule, comme /KEKW
  • tu choisis une taille, comme /KEKW:4x
  • tu mets plusieurs émotes dans le même message
  • tu écris du texte et des émotes dans le même message
/KEKW
Bonjour /KEKW
Salut /KEKW /OMEGALUL
/CATJAM:4x

Commandes utiles

/emotes

  • voir les émotes disponibles
  • chercher une émote par son nom
  • prévisualiser une émote
  • envoyer directement l'émote choisie

/emotes <nom> <taille> <texte>

  • nom pour choisir l'émote
  • taille pour choisir sa taille
  • texte pour ajouter un message avec l'émote

Exemple : /emotes nom:KEKW taille:4x texte:Salut

/info <message_id>

Cette commande sert à retrouver l'émote utilisée sur un message déjà renvoyé par le bot.

Exemple : /info message_id:1234567890
