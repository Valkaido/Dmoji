Comment utiliser le bot

Comment ecrire une emote

Tu peux utiliser l'un de ces formats :

  • /KEKW
  • /KEKW:1x
  • /KEKW:2x
  • /KEKW:3x
  • /KEKW:4x

Tu peux aussi ecrire du texte normal dans le meme message, par exemple Salut /KEKW.

Ce qui fonctionne

  • tu envoies une emote seule, comme /KEKW
  • tu choisis une taille, comme /KEKW:4x
  • tu mets plusieurs emotes dans le meme message
  • tu ecris du texte et des emotes dans le meme message
/KEKW
Bonjour /KEKW
Salut /KEKW /OMEGALUL
/CATJAM:4x

Ce qui bloque le bot

Commandes utiles

/emotes

  • voir les emotes disponibles
  • chercher une emote par son nom
  • previsualiser une emote
  • envoyer directement l'emote choisie

/emotes <nom> <taille> <texte>

  • nom pour choisir l'emote
  • taille pour choisir sa taille
  • texte pour ajouter un message avec l'emote

Exemple : /emotes nom:KEKW taille:4x texte:Salut

/info

Cette commande sert a retrouver l'emote utilisee sur un message deja renvoye par le bot.

Exemple : /info message_id:1234567890
